About This Assistant

Our voice-controlled web assistant empowers motor-impaired users to navigate websites using simple voice commands. With no installation needed, it runs entirely in the browser and responds in real-time.

Key Features

  • 🎤 Azure-powered speech-to-text
  • 🧠 Context-aware voice commands like “scroll down”, “click menu”
  • 🌐 Browser-based — no software or plugin installation

How It Works

Using Microsoft Azure’s Speech SDK, the assistant listens to voice input, transcribes it with deep learning models, and maps it to live DOM interactions — enabling seamless, hands-free control.
